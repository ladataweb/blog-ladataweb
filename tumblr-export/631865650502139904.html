[PowerBi] Atajos clave para codear en DAX

A muchos nos toca escribir código DAX diaria o semanalmente. Pasar tanto tiempo con la sección para escribir código de Power Bi te lleva a considerar mejoras y comodidades para las consultas que tienen gran tamaño. Cuando tu DAX supera las 10 lineas te comenzas a preocupar en caso que llegue un cambio para no estar reemplazando click por click o escribir muchas veces lo mismo.

En Power Bi se puede optimizar tu tiempo en DAX y ayudar tanto a ser más eficiente como a evitar errores de reemplazos. Todas éstas mágicas ayudas se llaman “Shortcuts” o atajos. En este artículo vamos a repasar las claves para hacer pequeñas magias en el texto con el teclado.

Atajos

Antes de ver los atajos voy a aclarar que solo funcionan en Power Bi. Por el momento no pueden usarlos en Power Pivot o SSAS. Para ordenarnos mejor voy a separar los atajos en dos tipos según mi criterio personal.

Clásicos

  • Ir a la línea…: Ctrl + G 
  • Borrar palabra: Ctrl + Del
  • Borrar una o muchas líneas: Ctrl + Shif + K. Para borrar muchas hay que seleccionarlas y luego Ctrl + Shif + K
  • Seleccionar una línea: Ctrl + i, sin importar donde este el cursor.
  • Insertar una línea arriba del cursor: Ctrl + Shift + Enter
  • Identificar delimitador inicial o final (paréntesis o corchetes) donde esta el cursor: Ctrl + Shift + \
  • Scrollear arriba o abajo: Ctrl + Fecha Arriba or Ctrl + Flecha Abajo
  • Sangría a la derecha: Ctrl + ]
  • Sangría a la izquierda: Ctrl + [
  • Insertar fila nueva abajo siguiendo sangría de código: Shift + Enter

Geniales

  • Comentar muchas lineas: Alt + Shift + A, Ctrl + KC o Ctrl + /
  • Descomentar muchas líneas: Ctrl + KU or Ctrl + /
  • Cambiar pestaña, mover el foco: Ctrl + M
  • Mover una linea arriba o abajo: Alt + Flecha Arriba/Abajo 
  • Expandir el cursor a lineas arriba o abajo para escribir en más de una linea al mismo tiempo: Ctrl + Alt + Flecha Arriba/Abajo 
  • Seleccionar palabra y también permite encontrar siguiente palabras idénticas a la posicionada si se vuelve a presionar: Ctrl + D. Es ideal para reemplazos.
  • Encontrar todas las palabras en el código idénticas a la posicionada por el cursor: Ctrl + Shift + L. Ideal para buscar y reemplazar todo.
  • Duplicar linea de código en linea de arriba/abajo: Alt + Shift + Flecha Arriba/Abajo

Como habrán visto muchos de ellos coinciden con los de Visual Studio Code. Espero que les haya servido y sea de utilidad.
