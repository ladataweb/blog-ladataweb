[Embed Power Bi] Incorporar Power Bi a una Web

Que mejor forma de dar a conocer datos que mostrarlos en tu web dentro de Power Bi. Para ello voy contarles sobre dos sencillas soluciones que nos ayudarán a hacerlo realidad. La diferencia entre ellas es la autenticación y la posibilidad de exportar datos. Analicemos entonces Embed y Publish to Web. Ambos casos se resuelven de formas semejantes. Tras seleccionar una de ellas, Power Bi muy amablemente nos concederá una dirección para embeber en el sitio como así también el código html de un objeto iframe que se utiliza para este fin.

Por un lado tenemos “Embed”. Es la forma elegida a niveles corporativos para incluir power bi a su intranet o sistema propio. De las dos formas, ésta es la más segura y será la que elegirán en caso de datos sensibles. El iframe de ésta opción requiere un inicio de sesión. Las credenciales que dan el acceso al Informe serán las de AD que cuenten con licencia Power Bi Pro. Son las mismas que permiten acceder a Service.

Por otro lado está la hermosa posibilidad que existe gratuitamente para usuarios Free. Por tan solo 0 pesos, siempre y cuando tengan una cuenta profesional de microsoft, es posible generar informes públicos para contenerlos en un sitio web. Hay que tener en cuenta que el enlace de “Publish To Web” es público, es decir que si alguien revisa el código html con “Inspeccionar elemento” de un navegador moderno, será capas de obtener el link de acceso. Éste modo tiene limitaciones RLS, SSAS, R, ArcGis. Respecto a “Exportar datos”, considero correcto que no sea permitido dado que los datos son públicos y de cierto modo así podemos restringir un poco la situación.
