Si hay algo que este mundo, lamentablemente, no está listo para reemplazar es el análisis de datos vía excel. La mayoría de nuestros usuarios finales de reportes que podamos construir en power bi, muy probablemente sean personas que conozcan y estén muy cómodos con manejo de datos por excel. No hablo sobre bellos gráficos de Power Bi, sino de chequeo de datos para preguntas extraordinarias que puedan surgir.

Aún teniendo el maravilloso self service Bi, la gente ama las tablas dinámicas de excel. Por ello, vamos a dedicar este post para enseñarles como conectar excel a la capa semántica (modelo tabular) de un Power Bi publicado en la nube para que esos usuarios pueden seguir cómodos sin tenernos como locos intentando ayudarlos infinitamente. 

Analizar con Excel

Hoy por hoy existe una única opción de conseguir esto y es desde Power Bi Service. Con "Analizar en Excel", puede traer conjuntos de datos de Power BI a Excel y luego verlos e interactuar con ellos a través de tablas dinámicas, gráficos, segmentaciones y otras características de Excel. Para usar Analizar en Excel, primero debe descargar la característica desde Power BI, instalarla y, luego, seleccionar uno o varios conjuntos de datos para usarlos en Excel.

Los pasos son muy sencillos. Conectado a Power Bi Service necesitamos descargar, la primera vez, un instalador para que excel pueda leer directamente del conjunto que seleccionemos. Para ello vamos dos opciones como muestran las siguientes imagenes:

En cualquier caso, Power BI detecta si tiene instalado Analizar en Excel y, si no es así, se le pedirá que lo descargue.

Una vez finalizada la descarga, ejecute el instalador (.msi) para instalar Analizar en Excel. El nombre del proceso de instalación no es Analizar en Excel, sino que será Proveedor OLE DB de Microsoft Analysis Services, tal como se muestra en la imagen siguiente, o un nombre similar.

Estamos listos para trabajar desde excel sobre nuestro conjunto de datos de Power Bi. Para lograr esto vamos a hacer el mismo proceso que la opción 2 seleccionado los tres puntos (1) del informe o conjunto de datos que deseamos y analizar en excel (2):

El servicio Power BI luego crea un archivo del conjunto de datos que está diseñado (y estructurado) para su uso con Analizar en Excel que tiene la extensión de archivo .ODC. Se crea el archivo y automáticamente se inicia un proceso de descarga en el explorador.

El archivo ya está asociado con Analizar en Excel por lo que, cuando seleccione o abra ese archivo .ODC, se iniciará Excel y el archivo .ODC se empezará a cargar automáticamente. Sin embargo, es probable que vea una advertencia sobre una amenaza de origen de datos externo:

Seleccione Habilitar para cargar el archivo .ODC para Analizar en Excel y Excel cargará el archivo.

El nombre de lo que necesitamos Power BI Publisher for Excel.

Con Microsoft Power BI Publisher para Excel, puede tomar instantáneas de sus datos más importantes en Excel, como tablas dinámicas, gráficos y rangos, y anclarlos a los paneles de Power BI. Veamos como funciona.

Lo primero que debemos hacer es descargar este plugin teniendo Excel 2013 o superior instalado. Enlace

Tras instalar este complemento aparecerá en nuestro excel una nueva pestaña llamada "Power Bi". Dicha pestaña es la que nos ayudará a conectarnos al conjunto de datos de Power Bi para construir nuestras tablas dinámicas y analizar datos.

A partir de ese momento, podrémos utilizar las nuevas opciones. Este nueva pestaña nos permitirá dos importantes comportamientos. Conectarse a un conjunto de datos o reporte y pinnear visualizaciones a paneles en Power Bi. Ésta última consta de llevar un gráfico de excel a un Panel en el área de trabajo selecta en Power Bi y puede experimentarla cuando quieras pero el post se centrará en la conexión del conjunto para análisis de datos.

Vamos a seleccionar Connect to Data y podremos elegir entre nuestras áreas de trabajo aquello que tengamos compartido. No estaría demás aclarar que solo podemos conectarnos a conjuntos que tengamos compartidos en Power Bi Service o de nuestra área personal.

Damos click en connect y estaremos listos para construir en excel. Veremos un menú a la derecha en una nueva página que nos permitirá utilizar los campos del conjunto de Power Bi. Tengamos en cuenta que para poder usar "Valores" en excel de las tablas solo podremos utilizar medidas de Power Bi.
